Smart Attendance System

🔹 Description
A simple Python-based attendance system that captures a user’s photo using a webcam and records their name with time in a CSV file.

🔹 Features
Capture image using webcam
Store attendance in CSV
Save user photo
Prevent duplicate entries

🔹 Technologies
Python, OpenCV, Pandas

🔹 How to Run
pip install opencv-python pandas
python main.py

🔹 Output
attendance.csv → stores name & time
attendence_marked/ → stores images
